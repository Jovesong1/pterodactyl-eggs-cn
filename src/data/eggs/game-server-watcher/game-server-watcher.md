# Game Server Watcher

## 来自 [Github](https://github.com/a-sync/game-server-watcher#readme)
一个简单的discord/telegram/slack机器人，可以托管在免费服务上，以时尚的方式监控您的游戏服务器和玩家。😎

## 服务器端口
GSW控制面板需要一个端口用于http流量。  
（可以是任何端口。）  

| 端口    | 默认值 |
|---------|---------|
| 网页     | 8081    |
