## 安装/系统需求

|  | 最低配置 | 推荐配置 |
|---------|---------|---------|
| 处理器 | *仅AMD64 300%* | *-* |
| 内存 | *18GiB* | *22GiB* |
| 存储 | *8GiB* | *10GiB* |
| 网络 | *任意* | *-* |
| 游戏所有权 | *不需要* | *-* |

## 服务器端口

运行服务器所需的端口，表格格式。

| 端口    | 默认值 |
|---------|---------|
| 游戏    | 7001    |
| 查询    | 7002   |

### 注意

7001是默认端口，但可以使用任何端口。
