# Redis

## 来自他们的[网站](https://redis.io/)

Redis是一个开源（BSD许可）的内存数据结构存储，用作数据库、缓存和消息代理。

## 最低内存警告

建议Redis至少有4GB的RAM

详情参见 <https://docs.redislabs.com/latest/rs/administering/designing-production/hardware-requirements/>
