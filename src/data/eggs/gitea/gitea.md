# Gitea

[Gitea](https://gitea.io/en-us/) 是一个用Go编写的社区管理的轻量级代码托管解决方案。它在MIT许可证下发布。

## 支持的版本
您可以在版本变量中设置版本。您可以使用以下值：

latest      - 最新稳定版本
nightly     - 最新Github主分支
version     - 根据此页面设置版本：https://dl.gitea.io/gitea

## 服务器端口

运行服务器所需的端口，以表格形式展示。
